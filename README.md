## Highlights

- 🍣 All the good stuff: `useHead()`, `useSeoMeta()`, `useScript()`, `useHeadSafe()`, `useServerHead()`,
- 🚀 Performant: Capo.js tag sorting, tiny (4.4kb gz), tree-shaken, etc.
- 🤝 Supports Vue, Angular, React, Svelte
- Used by [Nuxt](https://nuxt.com/)
